Clean — keywords

Cleans the keywords columns using the file keywords.txt, located in the same directory as the documents.csv file.

>>> from techminer import *
>>> directory = "/workspaces/techminer-api/data/"
>>> clean_keywords(directory)

2021-11-08 18:48:08 - INFO - Applying thesaurus to 'raw_keywords' column...

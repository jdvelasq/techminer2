Top Trending Topics

Extract and plot the top trending topics for the selected column using the average growth rate.

>>> from techminer2 import *
>>> directory = "/workspaces/techminer2/data/"
>>> file_name = "/workspaces/techminer2/sphinx/images/top_trending_topics.png"
>>> top_trending_topics(
...     column="author_keywords",
...     top_n=20,
...     directory=directory,
... ).savefig(file_name)

>>> top_trending_topics(
...     column='author_keywords',
...     directory=directory,
...     plot=False,
... ).head()
fintech                   21.5
financial technologies     4.0
financial inclusion        4.0
covid-19                   3.0
regulating                 2.5
Name: average_growth_rate, dtype: float64
